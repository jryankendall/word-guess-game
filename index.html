Guess The Word!

INSTRUCTIONS

A new game automatically begins upon first loading the page.

Press any letter on your keyboard to guess if that letter is in the word. If it's in the word at least once, all instance of that letter will be revealed. If that letter is not in the word at all, you will lose one Wrong Guess counter. Either way, the letter will be added to the list of guessed letters, and you won't be penalized for pressing that same letter.

If you guess an incorrect letter 8 times, you lose the current round. If you successfully guess all letters in the word before making 8 incorrect guesses you instead will win the round.

Press Spacebar to start a new round.
